# Vet-Cat - Ветеринарная клиника

Это фронтенд часть веб-сайта для вымышленной ветеринарной клиники "Vet-Cat". Сайт является многостраничным и предоставляет информацию об услугах клиники, специалистах, графике работы, а также позволяет записываться на прием.

## Технологии

Проект построен с использованием современного стека веб-технологий:

-   **Сборщик проекта:** [Vite](https://vitejs.dev/) - для быстрой разработки и сборки.
-   **Шаблонизатор HTML:** [Handlebars](https://handlebarsjs.com/) - используется через `vite-plugin-handlebars` для создания HTML-страниц из переиспользуемых частей (partials).
-   **Стилизация:** [SCSS (Sass)](https://sass-lang.com/) - для написания CSS с использованием переменных, миксинов и вложенности.
-   **JavaScript:** Современный JavaScript (ES6+) с модульной структурой.
-   **База данных:** [Firebase Firestore](https://firebase.google.com/docs/firestore) - используется в качестве бэкенда для хранения и получения данных (например, информация о врачах, услугах, статьях).
-   **Слайдеры:** [Swiper.js](https://swiperjs.com/) - для создания интерактивных слайдеров на страницах.
-   **Карты:** [Google Maps API](https://developers.google.com/maps) - для отображения местоположения клиник.
-   **Линтинг и форматирование:** ESLint, Prettier, Stylelint - для поддержания качества и единообразия кода.

```

├── dist/                # Папка для скомпилированных файлов (для продакшена)
├── pages/               # Исходные HTML файлы для каждой страницы
├── public/              # Статические ассеты (иконки, изображения)
├── src/

│   ├── js/              # JavaScript файлы
│   │   ├── base/        # Базовые скрипты (header, sidebar)
│   │   ├── composables/ # Переиспользуемые функции (хуки)
│   │   ├── partials/    # Логика для конкретных секций страниц
│   │   ├── widgets/     # Логика для сложных виджетов (слайдеры, попапы)
│   │   └── main.js      # Главный JS файл
│   └── styles/          # SCSS файлы
│       ├── base/        # Глобальные стили, шрифты
│       ├── pages/       # Стили для конкретных страниц
│       └── ...
├── .gitignore           # Файлы и папки, игнорируемые Git
├── firebase.js          # Конфигурация Firebase
├── package.json         # Зависимости и скрипты проекта
├── vite.config.js       # Конфигурация Vite
└── README.md            # Этот файл
